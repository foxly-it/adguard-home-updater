AdGuard Home Updater

Safe & automated updates for bare-metal installations

🚀 One-line Install
curl -fsSL https://install.foxly.de/install.sh | sudo bash
⚙️ Features

🔄 Auto Update

Checks and installs new versions automatically.

🛡 Rollback

Restores previous version on failure.

🔍 DNS Check

Ensures resolver is working after update.

🧪 Dry Run

Simulate updates safely.

🔧 Example
sudo adguard-update --dry-run
