Why this tool?

Adds mandatory release checksum verification, configuration validation, health checks, scheduling and rollback to supported bare-metal installations.

Highlights

What it does

Version check, binary backup, native service stop/start, DNS verification and rollback on failure.

Update process

Before any service interruption, the updater verifies the release and configuration.

Backups and rollback protect the running installation.

Settings and upgrades

Settings persist across installer upgrades.
