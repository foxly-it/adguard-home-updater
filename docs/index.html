AdGuard Home Updater

Automatisierte, sichere Updates für AdGuard Home – speziell für Bare-Metal Installationen und Homelabs.

Warum dieses Tool?

Der integrierte Updater von AdGuard Home funktioniert nicht auf allen Systemen zuverlässig – insbesondere auf ARM64-Systemen.

Dieses Script übernimmt den kompletten Update-Prozess inklusive Backup, Verifikation und Rollback.

Installation

curl -fsSL https://install.foxly.de/install.sh | sudo bash

Features

🔄 Automatische Updates

Erkennt neue Versionen und installiert sie sicher.

🛡 Rollback

Fehlgeschlagene Updates werden automatisch rückgängig gemacht.

🔍 DNS Health Check

Überprüft nach dem Update die Funktion des DNS.

🧪 Dry Run

Simuliert Updates ohne Änderungen am System.

Update Ablauf

Version prüfen → Release herunterladen → Binary extrahieren → AdGuard stoppen → Backup erstellen → Neue Version installieren → DNS prüfen → ggf. Rollback

Test (Dry Run)

sudo adguard-update --dry-run
